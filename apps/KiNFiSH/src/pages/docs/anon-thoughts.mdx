## AnonThoughts

**[AnonThoughts](https://anon-thoughts.vercel.app/)** - an anonymous social media platform that provides a confidential space for Ethiopians to freely express their thoughts, concerns, and experiences in their native language.

### AnonThoughts

The platform aims to create a culturally relevant and supportive environment where individuals can seek anonymous diagnosis and support on various issues from a culturally relevant perspective.

## Features

- Anonymous posting: Users can share their thoughts, concerns, or experiences anonymously without revealing their identities.
- Native language support: AnonSpeak focuses on providing support for Ethiopians to communicate in their native language, fostering a sense of cultural relevance and understanding.
- Anonymous diagnosis: Users can seek anonymous diagnosis and advice on personal, mental health, or other issues from professionals or experienced community members.
- Supportive community: AnonSpeak aims to build a supportive community where individuals can engage in discussions, offer support, and share insights from a culturally relevant perspective.
- Content moderation: The platform employs robust content moderation mechanisms to ensure a safe and respectful environment for users.
- Confidentiality: AnonSpeak takes privacy seriously and ensures that user identities and personal information are kept confidential.

### Solution

AnonThoughts provides the way to get a diagnosis as well support from other so that we can all elevate our mental status while staying around the web.

### How to use it ?

1. Clone the repository.
2. Install the dependencies with `npm install`.
3. Start the development server with `npm start`.
4. Mock the env file from .env.example
5. Open `http://localhost:3000` in your browser.
