## We-Farm

**[WeFarm](https://a2sv-notes.vercel.app/)** - Explore Internship Jobs and visually engaging learning slides cover data structures, algorithms, and other tech skills, providing a comprehensive understanding of concepts to help you grasp the skills you desire

### WeFarm

is an online platform that offers internship job opportunities and visually engaging learning slides to help individuals enhance their technical skills and gain a comprehensive understanding of concepts related to data structures, algorithms, and other tech skills.

## Features

- Browse and search for internship job listings from reputable companies.
- Apply for internships directly through the platform.
- Filter internships based on location, duration, industry, and more.
- Access visually engaging learning slides covering various technical topics.
- Interact with the learning slides for an immersive learning experience.
- Easily navigate through the platform with a user-friendly interface.

### Solution

WeFarm provides Internship Jobs and visually engaging learning slides cover data structures, algorithms, and other tech skills, providing a comprehensive understanding of concepts to help you grasp the skills you desire

### How to use it ?

1. Clone the repository.
2. Install the dependencies with `npm install`.
3. Start the development server with `npm start`.
4. Mock the env file from .env.example
5. Open `http://localhost:3000` in your browser.
